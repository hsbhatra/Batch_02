Projects to create as a Software Developer:

                1. Dive into APIs -
                    APIs are now used on a day-to-day basis, and they make work easy and fast pace. 
                        
                            Weather App, News App, etc. are some of the examples.
                        
                            


                2. Clone Apps - 
                    These clone apps will teach you how to make an end-to-end web-application, how the flow of the data actually works, how to build a beautiful UI, 
                    as many freshers aren't skilled in UI development.
                    
                            Instagram Clone, Facebook Clone, Spotify Clone, Netflix Clone, etc.
